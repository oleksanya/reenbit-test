# Test Chat App

Full-stack application built with Angular and Node.js

## Project Structure
```
/
├── client/         # Angular frontend
└── server/         # Node.js backend
```

## Setup Instructions

### Backend
```bash
cd server
npm install
npm run dev
```

### Frontend
```bash
cd client
npm install
npm start
```

## Technologies
- Frontend: Angular 19
- Backend: Node.js, Express
- Database: MongoDB
